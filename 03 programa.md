# ⚡ CURSO INTENSIVO: JAVASCRIPT PURO - "EL CEREBRO DE LA WEB"

**Docente:** José Manuel González Vargas
**Nivel:** Bachillerato (12-18 años)  
**Duración:** 10 Sesiones (90 min c/u)  
**Requisito:** HTML & CSS Básico  
**Enfoque:** Lógica Computacional & Manipulación del DOM

---

## 🧠 Filosofía del Curso

> "HTML es el sustantivo. CSS es el adjetivo. **JavaScript es el verbo**. Aquí es donde ocurre la magia, donde la web deja de ser un póster y se convierte en una aplicación viva."

---

## 📅 CRONOGRAMA DE MÓDULOS

### 🟢 FASE 1: FUNDAMENTOS (LA LÓGICA INVISIBLE)

### SESIÓN 1: LA CONSOLA Y LA MEMORIA (Variables)

**Objetivo:** Entender dónde vive el código y cómo guardar información.

* **0-20 min:** La Consola del Navegador (`F12`). Tu centro de mando.
* **20-50 min:** Variables: `let` (cajas abiertas) vs `const` (cajas fuertes). Tipos de datos: `String`, `Number`, `Boolean`.
* **50-80 min:** Operadores básicos: Sumar, restar y concatenar textos ("Hola" + " " + "Mundo").
* **Reto (10 min):** "Tu ID Digital": Crear variables con nombre, edad y escuela, y mostrarlas en una frase en consola.

### SESIÓN 2: TOMANDO DECISIONES (Condicionales)

**Objetivo:** Que el código reaccione de forma diferente según la situación.

* **0-30 min:** Comparadores: `>`, `<`, `===`, `!==`. La verdad y la mentira.
* **30-70 min:** La bifurcación: `if`, `else if`, `else`.

* **70-90 min:** **Reto:** "El Portero de Discoteca": Un script que pregunte la edad (variable) y diga en consola si puedes pasar o no.

### SESIÓN 3: LAS MÁQUINAS (Funciones Básicas)

**Objetivo:** Encapsular código para no repetirnos (DRY - Don't Repeat Yourself).

* **0-30 min:** Anatomía de una función: `function nombre() {}`.
* **30-60 min:** Parámetros y Argumentos: Alimentando la máquina.
* **60-90 min:** **Reto:** "La Calculadora Humana": Crear una función `saludar(nombre)` y una función `sumar(a, b)` que den resultados distintos.

### SESIÓN 4: LISTAS Y BUCLES (Arrays & Loops)

**Objetivo:** Manejar grandes cantidades de datos sin escribir mil líneas.

* **0-30 min:** Arrays `[]`: Listas de cosas (índices que empiezan en 0).
* **30-60 min:** El bucle `for`: Repetir una acción automáticamente.
* **60-90 min:** **Reto:** "El Pase de Lista": Crear un array con 5 nombres de amigos e imprimirlos uno por uno usando un bucle.

---

### 🔵 FASE 2: EL DOM (CONECTANDO CON LA REALIDAD)

### SESIÓN 5: TOCANDO EL HTML (Selectores)

**Objetivo:** Romper la barrera entre JS y HTML.

* **0-30 min:** ¿Qué es el DOM? (Document Object Model). El árbol genealógico de la web.

* **30-60 min:** El gancho: `document.getElementById()` y `document.querySelector()`.
* **60-90 min:** Manipulación: Cambiar `.innerHTML` y estilos `.style` desde JS.
* **Reto:** "El Hacker": Cambiar el título `h1` de la página y el color de fondo usando solo JS.

### SESIÓN 6: ESCUCHANDO AL USUARIO (Eventos I)

**Objetivo:** Hacer la página interactiva. Acción -> Reacción.

* **0-30 min:** El concepto de "Event Listener". Estar a la espera.
* **30-60 min:** El evento `click`. Hacer que un botón dispare una función.
* **60-90 min:** **Reto:** "Interruptor de Luz": Un botón que al hacer clic cambie el fondo de blanco a negro (Modo Oscuro básico).

### SESIÓN 7: LÓGICA VISUAL (ClassList)

**Objetivo:** Manipular estilos de forma profesional (sin ensuciar el HTML).

* **0-30 min:** Por qué no usar `.style` directo.
* **30-60 min:** `classList.add`, `.remove` y `.toggle`.
* **60-90 min:** **Reto:** "Menú Hamburguesa": Crear un botón que muestre y oculte un menú lateral usando `toggle`.

---

### 🟣 FASE 3: PROYECTOS APLICADOS

### SESIÓN 8: CAPTURANDO DATOS (Inputs)

**Objetivo:** Leer lo que el usuario escribe.

* **0-30 min:** Leer el `.value` de un input de texto.
* **30-60 min:** Validar datos: ¿El usuario escribió algo o dejó el campo vacío?
* **60-90 min:** **Reto:** "El Saludador VIP": Un input para poner tu nombre y un botón que al pulsarlo escriba "Bienvenido, [Nombre]" en un `h2`.

### SESIÓN 9: MATEMÁTICAS DINÁMICAS (Lógica + DOM)

**Objetivo:** Crear una herramienta funcional real.

* **0-30 min:** Transformar texto a número (`parseInt`, `parseFloat`).
* **30-80 min:** **Mini-Proyecto:** "Calculadora de Promedio".
  * 3 Inputs (Calificaciones).
  * 1 Botón (Calcular).
  * Lógica: Sumar y dividir.
  * Output: Mostrar el resultado y cambiar de color si está reprobado (<60) o aprobado.
* **80-90 min:** Debugging: Qué hacer cuando sale `NaN`.

### SESIÓN 10: EL GRAN JUEGO (Proyecto Final)

**Objetivo:** Integración total (HTML + CSS + JS).

* **0-20 min:** Planteamiento del problema: Lógica del juego.
* **20-80 min:** **Proyecto:** "Adivina el Número".
  * La computadora elige un número al azar (`Math.random`).
  * El usuario ingresa un número.
  * JS compara y dice "Más alto", "Más bajo" o "¡Ganaste!".
  * Contador de intentos.
* **80-90 min:** Presentación y cierre del curso.
