Sesión 6: Escuchando al Usuario

Aprendiendo a hacer páginas interactivas con Event Listeners

¿Qué son los Eventos?

En el mundo de la programación web, un evento es cualquier acción que ocurre en la página, como hacer clic en un botón, mover el mouse, presionar una tecla, o incluso cuando la página termina de cargar.

JavaScript nos permite escuchar estos eventos y responder a ellos ejecutando código. Esto es lo que hace que las páginas web sean interactivas.

Tipos comunes de eventos:

  • click - Cuando haces clic en un elemento
  • dblclick - Cuando haces doble clic
  • mouseover - Cuando el mouse pasa por encima
  • mouseout - Cuando el mouse sale del elemento
  • keydown - Cuando presionas una tecla
  • keyup - Cuando sueltas una tecla
  • submit - Cuando envías un formulario
  • load - Cuando la página termina de cargar
¡Recuerda! Los eventos son la forma en que JavaScript interactúa con el usuario. Sin eventos, las páginas web serían estáticas y aburridas.

Escuchando Eventos

Hay varias formas de manejar eventos en JavaScript, pero la más moderna y recomendada es usando addEventListener().

Sintaxis básica:

elemento.addEventListener('tipoDeEvento', funcionAEjecutar);

Por ejemplo, para hacer que un botón muestre un mensaje al hacer clic:

// 1. Seleccionar el botón
let boton = document.getElementById('btnSaludo');

// 2. Agregar un event listener para el evento 'click'
boton.addEventListener('click', function() {
    alert('¡Hola! Gracias por hacer clic.');
});

El Objeto Event

Cada vez que ocurre un evento, el navegador crea un objeto event que contiene información sobre lo que sucedió. Este objeto se pasa automáticamente a la función manejadora del evento.

elemento.addEventListener('click', function(evento) {
    // 'evento' es el objeto que contiene información sobre el evento
    console.log('Tipo de evento:', evento.type);
    console.log('Elemento clickeado:', evento.target);
    console.log('Coordenadas X:', evento.clientX, 'Y:', evento.clientY);
});

Eventos de Teclado

Los eventos de teclado nos permiten detectar cuando el usuario presiona teclas. Los eventos principales son:

  • keydown: Se dispara cuando una tecla se presiona
  • keyup: Se dispara cuando una tecla se suelta
  • keypress: Se dispara cuando se genera un carácter (obsoleto, no recomendado)

¿Tienes dudas sobre eventos en JavaScript? ¡Pregúntame!

Reto: Interruptor de Luz

Crea un botón que cambie el tema de la página entre claro y oscuro cuando se haga clic en él.

Área del Reto

Tu solución aparecerá aquí:

Pista: Usa classList.toggle() para alternar una clase que cambie los colores de la página. Define los estilos para el modo oscuro en el CSS.

Solución:

// 1. Seleccionar el botón
let boton = document.getElementById('btnToggleTheme');

// 2. Agregar un event listener para el evento 'click'
boton.addEventListener('click', function() {
    // 3. Alternar la clase 'dark-mode' en el body
    document.body.classList.toggle('dark-mode');
    
    // Opcional: Cambiar el texto del botón
    if (document.body.classList.contains('dark-mode')) {
        boton.textContent = 'Modo Claro';
    } else {
        boton.textContent = 'Modo Oscuro';
    }
});

Y en el CSS:

.dark-mode {
    background-color: #1a1a1a;
    color: #ffffff;
}

.dark-mode .content {
    background-color: #2d2d2d;
}

/* Agrega más estilos según sea necesario */

Resumen de la sesión

  • Los eventos son acciones que ocurren en la página web, como hacer clic o presionar una tecla.
  • Usamos addEventListener() para escuchar eventos y ejecutar código cuando ocurren.
  • El objeto event contiene información sobre el evento que ocurrió.
  • Los eventos comunes incluyen click, mouseover, keydown, keyup, y submit.
  • Podemos prevenir el comportamiento predeterminado de los eventos con event.preventDefault().
  • Los eventos son fundamentales para crear páginas web interactivas que respondan a las acciones del usuario.
