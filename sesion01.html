Sesión 1: La Consola y la Memoria

Aprende a usar la consola y variables en JavaScript

📱 La Consola del Navegador

La consola es como el "cuaderno de notas" de los programadores web. Es donde podemos ver mensajes, errores y resultados de nuestro código.

💡 Cómo abrir la consola:
  • Windows/Linux: Presiona F12 o Ctrl + Shift + I
  • Mac: Presiona Cmd + Option + I

Una vez abierta, ve a la pestaña "Console". Aquí es donde verás los resultados de tu código.

Tu primer comando en la consola

Escribe lo siguiente en la consola y presiona Enter:

console.log("¡Hola, mundo!");

¡Felicidades! Acabas de ejecutar tu primera línea de JavaScript. Deberías ver el mensaje "¡Hola, mundo!" en la consola.

💾 Variables: Las cajas de la memoria

Imagina que las variables son como cajas donde puedes guardar cosas. En JavaScript, usamos palabras clave para crear variables:

// Crear una variable que puede cambiar (caja abierta) let nombre = "Ana"; // Crear una variable que NO puede cambiar (caja fuerte) const PI = 3.1416;

Tipos de datos básicos

// 1. String (texto) - van entre comillas let saludo = "Hola"; // 2. Number (números) - sin comillas let edad = 15; // 3. Boolean (verdadero/falso) - sin comillas let esEstudiante = true;
💡 Regla de oro: Usa const por defecto, y solo usa let si sabes que el valor va a cambiar.

Operadores básicos

Podemos hacer operaciones matemáticas y con texto:

// Suma de números let suma = 5 + 3; // Resultado: 8 // Concatenación de textos let nombreCompleto = "Ana" + " " + "Pérez"; // Resultado: "Ana Pérez" // Suma de número y texto (¡cuidado!) let resultado = "5" + 3; // Resultado: "53" (¡es un texto!)

🧩 Actividad Práctica

Intenta lo siguiente en la consola:

  1. Crea una variable llamada materia con el valor "Matemáticas"
  2. Crea una variable llamada calificacion con el valor 9.5
  3. Crea una constante llamada ESCUELA con el nombre de tu escuela
  4. Usa console.log() para mostrar: "En [ESCUELA], mi calificación en [MATERIA] es [CALIFICACION]"
const ESCUELA = "Mi Escuela"; let materia = "Matemáticas"; let calificacion = 9.5; console.log("En " + ESCUELA + ", mi calificación en " + materia + " es " + calificacion);

📝 Resumen de la Sesión

  • La consola del navegador es una herramienta esencial para programar en JavaScript.
  • Las variables son como cajas donde guardamos información.
  • Usamos let para variables que pueden cambiar y const para constantes.
  • Los tipos de datos básicos son: texto (String), números (Number) y booleanos (true/false).
  • Podemos hacer operaciones con números y concatenar textos con el operador +.
