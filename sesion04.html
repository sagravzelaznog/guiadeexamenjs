Sesión 4: Listas y Bucles

Aprendiendo a manejar múltiples datos y a automatizar tareas repetitivas

¿Qué son las listas (arrays) en JavaScript?

Imagina que tienes una caja de lápices de colores. En lugar de tener variables separadas para cada color, puedes guardarlos todos juntos en una lista. En JavaScript, a estas listas las llamamos arrays.

0
Rojo
1
Azul
2
Verde
3
Amarillo

Creando un array

Para crear un array, usamos corchetes [] y separamos los elementos con comas:

// Crear un array de colores
let colores = ["rojo", "azul", "verde", "amarillo"];

// Los arrays pueden contener cualquier tipo de dato
let datosMixtos = ["texto", 42, true, null];
¡Recuerda! Los índices de los arrays en JavaScript comienzan en 0. El primer elemento está en la posición 0, el segundo en la 1, y así sucesivamente.

Accediendo a los elementos

Podemos acceder a los elementos de un array usando su índice entre corchetes:

let frutas = ["manzana", "plátano", "naranja"];
console.log(frutas[0]);  // Muestra: manzana
console.log(frutas[1]);  // Muestra: plátano
console.log(frutas[2]);  // Muestra: naranja

Propiedades y métodos útiles de los arrays

Propiedad length

Nos dice cuántos elementos tiene el array:

let numeros = [10, 20, 30, 40, 50];
console.log(numeros.length);  // Muestra: 5

Métodos comunes

  • push(): Añade un elemento al final del array
  • pop(): Elimina el último elemento del array
  • unshift(): Añade un elemento al principio del array
  • shift(): Elimina el primer elemento del array

Bucles: Repitiendo acciones

Imagina que tienes que saludar a cada persona en una lista de nombres. En lugar de escribir un saludo para cada uno, podemos usar un bucle.

Bucle for

El bucle for nos permite repetir un bloque de código un número específico de veces:

// Estructura básica de un for
for (inicialización; condición; actualización) {
    // código que se repite
}

// Ejemplo: Contar del 1 al 5
for (let i = 1; i <= 5; i++) {
    console.log(i);
}

Recorriendo un array con for

Podemos usar un bucle for para recorrer todos los elementos de un array:

let frutas = ["manzana", "plátano", "naranja"];

for (let i = 0; i < frutas.length; i++) {
    console.log("Me gusta la " + frutas[i]);
}

Recorriendo un array

Array: ["manzana", "plátano", "naranja"]

Bucle for...of

Una forma más sencilla de recorrer arrays (introducido en ES6):

let frutas = ["manzana", "plátano", "naranja"];

for (let fruta of frutas) {
    console.log("Me gusta la " + fruta);
}

Resumen de la sesión

  • Los arrays son listas que nos permiten almacenar múltiples valores en una sola variable.
  • Los elementos de un array tienen índices que comienzan en 0.
  • Podemos acceder a los elementos usando su índice: miArray[0].
  • La propiedad length nos da el número de elementos en un array.
  • Los bucles for nos permiten repetir un bloque de código varias veces.
  • Podemos usar bucles para recorrer arrays y realizar operaciones con cada elemento.
  • El bucle for...of es una forma más sencilla de recorrer arrays.
