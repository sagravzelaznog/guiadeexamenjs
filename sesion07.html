Sesión 7: Lógica Visual (ClassList)

Aprendiendo a manipular estilos de forma profesional con classList

¿Por qué no usar .style directo?

En la sesión anterior vimos cómo modificar estilos directamente con elemento.style.propiedad, pero esta no es la mejor práctica por varias razones:

❌ Usando .style

// Código difícil de mantener
elemento.style.backgroundColor = '#e74c3c';
elemento.style.color = 'white';
elemento.style.padding = '15px';
elemento.style.borderRadius = '8px';

// Para quitar un estilo, hay que ponerlo en blanco
elemento.style.backgroundColor = '';
Caja de ejemplo

✅ Usando classList

/* En el CSS */
.mi-estilo {
  background-color: #e74c3c;
  color: white;
  padding: 15px;
  border-radius: 8px;
}

// En JavaScript
elemento.classList.add('mi-estilo');
// Para quitar la clase
elemento.classList.remove('mi-estilo');
Caja de ejemplo
¡Ventajas de usar classList!
  • Más limpio y organizado: Separas la estructura (HTML), el estilo (CSS) y el comportamiento (JS).
  • Más fácil de mantener: Los cambios de diseño se hacen en el CSS, no en el JS.
  • Mejor rendimiento: El navegador optimiza mejor los cambios de clase que los estilos individuales.
  • Más poderoso: Puedes aplicar múltiples estilos a la vez con una sola clase.

Métodos de classList

El objeto classList tiene varios métodos útiles para trabajar con clases CSS:

1. classList.add()

Agrega una o más clases a un elemento.

elemento.classList.add('clase1');
// Múltiples clases
elemento.classList.add('clase2', 'clase3');
Haz clic en "Probar"
2. classList.remove()

Elimina una o más clases de un elemento.

elemento.classList.remove('clase1');
// Múltiples clases
elemento.classList.remove('clase2', 'clase3');
Esta caja es roja
3. classList.toggle()

Alterna una clase (la agrega si no existe, la quita si existe).

// Alterna la clase 'activo'
elemento.classList.toggle('activo');
// Con un segundo parámetro booleano
elemento.classList.toggle('activo', true); // Siempre agrega
Haz clic en "Alternar" varias veces
4. classList.contains()

Comprueba si un elemento tiene una clase específica.

// Devuelve true o false
if (elemento.classList.contains('activo')) {
  // El elemento tiene la clase 'activo'
}
Esta caja no tiene clases especiales

Consejo profesional: Puedes encadenar múltiples métodos de classList para realizar varias operaciones a la vez:
elemento.classList .remove('clase-antigua') .add('clase-nueva') .toggle('activo');

Reto: Menú Hamburguesa

¡Es hora de poner en práctica lo aprendido! Crea un menú hamburguesa que se muestre y oculte al hacer clic en el botón.

Instrucciones:

  1. Ya tienes el HTML y CSS básico del menú hamburguesa.
  2. Usa classList.toggle() para mostrar/ocultar el menú lateral cuando se haga clic en el botón.
  3. El menú debe deslizarse desde la izquierda.
  4. El botón debe cambiar a una "X" cuando el menú está abierto (usa la clase active).
  5. El overlay debe aparecer/desaparecer junto con el menú.

Pista:

Usa classList.toggle() en el menú lateral, el botón y el overlay para alternar la clase active que ya está definida en el CSS.

El código debería verse así:

boton.addEventListener('click', function() {
  menu.classList.toggle('active');
  boton.classList.toggle('active');
  overlay.classList.toggle('active');
});

Solución completa:

Aquí está el código JavaScript para implementar el menú hamburguesa:

// Seleccionar elementos
const hamburgerBtn = document.getElementById('hamburgerBtn');
const sidebarMenu = document.getElementById('sidebarMenu');
const menuOverlay = document.getElementById('menuOverlay');

// Función para alternar el menú
function toggleMenu() {
  hamburgerBtn.classList.toggle('active');
  sidebarMenu.classList.toggle('active');
  menuOverlay.classList.toggle('active');
}

// Event listeners
hamburgerBtn.addEventListener('click', toggleMenu);
menuOverlay.addEventListener('click', toggleMenu);

// Cerrar menú al hacer clic en un enlace
document.querySelectorAll('.sidebar-menu a').forEach(link => {
  link.addEventListener('click', toggleMenu);
});

Resumen de la sesión

  • classList es una forma poderosa de manipular clases CSS desde JavaScript.
  • Métodos principales:
    • add() - Agrega una o más clases.
    • remove() - Elimina una o más clases.
    • toggle() - Alterna una clase (agrega si no existe, quita si existe).
    • contains() - Comprueba si un elemento tiene una clase específica.
  • Ventajas de usar classList sobre style directo:
    • Mejor organización del código (separación de preocupaciones).
    • Más fácil de mantener y actualizar.
    • Mejor rendimiento para múltiples cambios de estilo.
    • Más legible y menos propenso a errores.
  • Los métodos de classList se pueden encadenar para realizar múltiples operaciones a la vez.
