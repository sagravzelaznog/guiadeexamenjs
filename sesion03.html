⚙️ Sesión 3: Las Máquinas (Funciones Básicas)

Aprende a crear y usar funciones para organizar y reutilizar tu código

Objetivo de la Sesión

Entender cómo crear y utilizar funciones en JavaScript para organizar el código y evitar repeticiones (principio DRY: Don't Repeat Yourself).

🤖 ¿Qué es una función?

Una función es como una máquina que realiza una tarea específica. Le das una entrada (parámetros), realiza algún procesamiento y te devuelve un resultado (valor de retorno).

Entrada (Parámetros) → Función → Salida (Valor de retorno)
💡 Por qué usar funciones:
  • Reutilización de código: Escribe una vez, usa muchas veces.
  • Organización: Divide tu código en partes más pequeñas y manejables.
  • Mantenimiento: Si necesitas hacer un cambio, solo lo haces en un lugar.

📝 Sintaxis de una Función

function saludar (nombre) { } console.log(`¡Hola, ${nombre}!`); return resultado;

1. Función sin parámetros ni retorno

// Definición de la función function saludar() { console.log("¡Hola, mundo!"); } // Llamada a la función saludar(); // Muestra: ¡Hola, mundo!

2. Función con parámetros

function saludar(nombre) { console.log(`¡Hola, ${nombre}!`); } saludar("Ana"); // Muestra: ¡Hola, Ana! saludar("Carlos"); // Muestra: ¡Hola, Carlos!

3. Función con retorno

function sumar(a, b) { return a + b; } let resultado = sumar(5, 3); console.log(resultado); // Muestra: 8 // También podemos usar el resultado directamente console.log(sumar(10, 20)); // Muestra: 30
💡 Buenas prácticas:
  • Usa nombres descriptivos para las funciones (verbos como calcular, obtener, verificar).
  • Una función debe hacer una sola cosa y hacerla bien.
  • Si una función es muy larga, considera dividirla en funciones más pequeñas.

🧮 Calculadora de Operaciones Básicas

Vamos a crear una calculadora simple usando funciones para cada operación.

// Funciones de la calculadora function sumar(a, b) { return a + b; } function restar(a, b) { return a - b; } function multiplicar(a, b) { return a * b; } function dividir(a, b) { if (b === 0) { return "Error: No se puede dividir por cero"; } return a / b; } // Función principal que usa las demás function calcular() { const num1 = parseFloat(document.getElementById('num1').value); const num2 = parseFloat(document.getElementById('num2').value); const operacion = document.getElementById('operacion').value; let resultado; // Usamos un switch para seleccionar la operación switch(operacion) { case 'sumar': resultado = sumar(num1, num2); break; case 'restar': resultado = restar(num1, num2); break; case 'multiplicar': resultado = multiplicar(num1, num2); break; case 'dividir': resultado = dividir(num1, num2); break; default: resultado = "Operación no válida"; } // Mostramos el resultado document.getElementById('resultadoCalculadora').textContent = `Resultado: ${num1} ${getSimboloOperacion(operacion)} ${num2} = ${resultado}`; // Añadimos un poco de estilo const resultadoElemento = document.getElementById('resultadoCalculadora'); resultadoElemento.style.color = 'white'; resultadoElemento.style.backgroundColor = '#0066cc'; resultadoElemento.style.padding = '10px'; resultadoElemento.style.borderRadius = '4px'; } // Función auxiliar para obtener el símbolo de la operación function getSimboloOperacion(operacion) { const simbolos = { 'sumar': '+', 'restar': '-', 'multiplicar': '×', 'dividir': '÷' }; return simbolos[operacion] || '?'; }

🧩 Actividad Práctica: La Calculadora Humana

Crea dos funciones siguiendo las instrucciones:

1. Función saludar(nombre)

Crea una función que tome un parámetro nombre y devuelva un saludo personalizado.

2. Función sumar(a, b)

Crea una función que tome dos números como parámetros y devuelva su suma.

// Ejemplo de cómo debería funcionar console.log(saludar("Ana")); // Debería mostrar: "¡Hola, Ana! ¿Cómo estás?" console.log(sumar(7, 3)); // Debería devolver: 10
// Solución para la función saludar function saludar(nombre) { return `¡Hola, ${nombre}! ¿Cómo estás?`; } // Solución para la función sumar function sumar(a, b) { return a + b; } // Probando las funciones console.log(saludar("Ana")); // Muestra: ¡Hola, Ana! ¿Cómo estás? console.log(sumar(7, 3)); // Muestra: 10

📝 Resumen de la Sesión

  • Las funciones son bloques de código reutilizables que realizan una tarea específica.
  • Una función puede recibir parámetros (entrada) y devolver un valor (salida).
  • La sintaxis básica es: function nombreFuncion(parametros) { ... }
  • Usamos return para que una función devuelva un valor.
  • Las funciones nos ayudan a organizar el código y evitar repeticiones (principio DRY).
  • Es buena práctica que una función haga una sola cosa y la haga bien.
