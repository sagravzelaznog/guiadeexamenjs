1. Introducción a los Inputs

En esta sesión aprenderás a capturar datos que los usuarios ingresan en formularios web. Esto es esencial para crear aplicaciones interactivas que respondan a la entrada del usuario.

¿Por qué son importantes los inputs?

Los inputs permiten a los usuarios interactuar con tu sitio web. Sin ellos, las páginas web serían estáticas y no podrías hacer cosas como:

  • Iniciar sesión en sitios web
  • Realizar búsquedas
  • Enviar mensajes
  • Completar formularios de contacto
  • Y mucho más...

3. Validación de Formularios

La validación es crucial para asegurar que los datos ingresados por los usuarios sean correctos antes de enviarlos al servidor.

Validación con HTML5

HTML5 incluye atributos de validación integrados:

  • required - El campo es obligatorio
  • minlength - Longitud mínima del texto
  • maxlength - Longitud máxima del texto
  • min y max - Valores mínimo y máximo para números
  • pattern - Expresión regular para validar el formato
  • type="email" - Valida que sea un correo electrónico válido

4. Eventos de Input

JavaScript proporciona varios eventos para manejar la interacción del usuario con los inputs:

  • input - Se dispara cada vez que el valor de un input cambia
  • change - Se dispara cuando el usuario termina de cambiar el valor (al quitar el foco)
  • focus - Cuando el input recibe el foco
  • blur - Cuando el input pierde el foco
  • keydown / keyup - Cuando se presiona/suelta una tecla

🏆 Reto: El Saludador VIP

Crea un formulario con un campo para el nombre y un botón. Al hacer clic en el botón, debe aparecer un mensaje de bienvenida personalizado en un elemento de la página.

Requisitos:

  1. Un campo de texto para que el usuario ingrese su nombre.
  2. Un botón que diga "Saludar".
  3. Un elemento donde se mostrará el mensaje de bienvenida.
  4. El mensaje debe decir: "¡Bienvenido/a, [Nombre]! Esperamos que disfrutes el curso de JavaScript."
  5. Si el campo está vacío, muestra un mensaje de error.

Pista: Usa document.getElementById() para obtener los elementos del DOM, y el evento click en el botón para ejecutar una función que lea el valor del input y actualice el mensaje.

Ejemplo de cómo obtener el valor de un input:

const nombre = document.getElementById('nombre-input').value;

Y para actualizar el texto de un elemento:

document.getElementById('mensaje').textContent = 'Nuevo mensaje';
