Sesión 5: Tocando el HTML

Aprendiendo a manipular la página web con JavaScript

¿Qué es el DOM?

El Document Object Model (Modelo de Objetos del Documento) es una representación en forma de árbol de todos los elementos de una página web. Imagínalo como el árbol genealógico de tu página.

Estructura del DOM

- <html>
- <head>
<title> Mi Página </title>
</head>
- <body>
<h1> ¡Hola Mundo! </h1>
<p> Bienvenido a mi página web. </p>
</body>
</html>

Cada caja en este árbol es un nodo. Los nodos pueden ser elementos, atributos o texto. Con JavaScript, podemos navegar por este árbol y modificar cualquier parte de él.

¡Recuerda! El DOM es como un puente entre tu código JavaScript y la página web. Cuando cambias algo en el DOM, la página se actualiza automáticamente.

Seleccionando elementos

Para modificar elementos HTML, primero necesitamos seleccionarlos. Aquí están los métodos más comunes:

1. Por ID: getElementById()

Selecciona un elemento por su atributo id (debe ser único en la página).

// HTML: <div id="miElemento">Hola</div>
let elemento = document.getElementById('miElemento');
console.log(elemento.textContent); // Muestra: Hola

2. Por selector CSS: querySelector() y querySelectorAll()

Permite seleccionar elementos usando selectores CSS.

// Selecciona el PRIMER elemento que coincida
let primerParrafo = document.querySelector('p');

// Selecciona TODOS los elementos que coincidan
let todosParrafos = document.querySelectorAll('p');

// Selecciona por clase
let items = document.querySelectorAll('.item');

  • #caja2 - El elemento con ID "caja2"
  • div - Todos los elementos div

  • p:first-child - Primer párrafo dentro de su contenedor

Modificando elementos

Una vez que seleccionamos un elemento, podemos modificarlo de varias formas:

1. Cambiar el contenido: textContent e innerHTML

// Cambiar texto plano (seguro contra inyección de HTML)
elemento.textContent = 'Nuevo texto';

// Cambiar HTML (permite etiquetas HTML)
elemento.innerHTML = '<strong>Texto en negrita</strong>';

// Obtener contenido
let texto = elemento.textContent;
let html = elemento.innerHTML;

2. Cambiar atributos: getAttribute() y setAttribute()

// Obtener un atributo
let ruta = imagen.getAttribute('src');

// Establecer un atributo
imagen.setAttribute('src', 'nueva-imagen.jpg');
imagen.setAttribute('alt', 'Descripción de la imagen');

// Atributos directos (solo para atributos estándar)
imagen.src = 'nueva-imagen.jpg';
boton.disabled = true;

3. Cambiar estilos: style

// Cambiar estilos directamente
elemento.style.color = 'red';
elemento.style.backgroundColor = 'yellow';
elemento.style.fontSize = '20px';

// Nota: Las propiedades con guiones se escriben en camelCase:
// background-color -> backgroundColor
// font-size -> fontSize
¡Cuidado! Modificar estilos directamente con elemento.style no es la mejor práctica. Es mejor usar clases CSS y alternarlas con classList (lo veremos en la siguiente sesión).

Resumen de la sesión

  • El DOM es la representación en forma de árbol de todos los elementos de una página web.
  • Podemos seleccionar elementos usando getElementById(), querySelector() y querySelectorAll().
  • Podemos modificar el contenido de los elementos con textContent (solo texto) o innerHTML (permite HTML).
  • Los atributos se pueden leer y modificar con getAttribute() y setAttribute().
  • Podemos cambiar estilos directamente con la propiedad style, pero es mejor práctica usar clases CSS.
  • Con estas herramientas, ¡podemos modificar cualquier aspecto de una página web en tiempo real!
