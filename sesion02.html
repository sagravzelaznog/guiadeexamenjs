🎯 Sesión 2: Tomando Decisiones (Condicionales)

Aprende a hacer que tu código tome decisiones como un experto

Objetivo de la Sesión

Entender cómo hacer que los programas tomen decisiones basadas en condiciones usando sentencias condicionales en JavaScript.

🔍 Comparadores: La base de las decisiones

Antes de tomar decisiones, necesitamos saber comparar valores. En JavaScript usamos operadores de comparación:

Operador Significado Ejemplo Resultado
== Igual a (valor) 5 == 5 true
=== Igual en valor y tipo 5 === '5' false
!= No igual (valor) 5 != 3 true
!== No igual en valor o tipo 5 !== '5' true
> Mayor que 10 > 5 true
< Menor que 10 < 5 false
>= Mayor o igual que 10 >= 10 true
<= Menor o igual que 10 <= 5 false
💡 Buenas prácticas: Siempre usa === y !== en lugar de == y != para evitar comportamientos inesperados.

🎚️ Estructuras Condicionales

1. if - La decisión simple

Ejecuta un bloque de código si una condición es verdadera.

//let edad = 16; // if (edad >= 18) { console.log("¡Puedes votar!"); }

2. if...else - Dos caminos

Ejecuta un bloque si la condición es verdadera y otro si es falsa.

let hora = 14; // 2 PM if (hora < 12) { console.log("Buenos días"); } else { console.log("Buenas tardes"); }

3. if...else if...else - Múltiples condiciones

Permite evaluar múltiples condiciones en secuencia.

let calificacion = 85; if (calificacion >= 90) { console.log("Excelente: A"); } else if (calificacion >= 80) { console.log("Muy bien: B"); } else if (calificacion >= 70) { console.log("Bien: C"); } else if (calificacion >= 60) { console.log("Suficiente: D"); } else { console.log("Reprobado: F"); }
💡 Orden de las condiciones: Las condiciones se evalúan en orden. Coloca primero las condiciones más específicas.

4. Operador Ternario (?:) - If en una línea

Una forma concisa de escribir condiciones simples.

let edad = 20; let puedeVotar = (edad >= 18) ? "Puede votar" : "No puede votar"; console.log(puedeVotar); // "Puede votar"

🎮 Demo Interactivo: El Portero de Discoteca

Vamos a crear un programa que determine si puedes entrar a una discoteca según tu edad.

🧩 Actividad Práctica

Crea un programa que determine la estación del año según el mes (1-12).

  1. Crea una variable mes con un número del 1 al 12.
  2. Usa condicionales para determinar la estación del año:
    • Diciembre, Enero, Febrero: Invierno
    • Marzo, Abril, Mayo: Primavera
    • Junio, Julio, Agosto: Verano
    • Septiembre, Octubre, Noviembre: Otoño
  3. Muestra el resultado con console.log().
let mes = 5; // Puedes cambiar este valor let estacion; if (mes === 12 || mes === 1 || mes === 2) { estacion = "Invierno"; } else if (mes >= 3 && mes <= 5) { estacion = "Primavera"; } else if (mes >= 6 && mes <= 8) { estacion = "Verano"; } else if (mes >= 9 && mes <= 11) { estacion = "Otoño"; } else { estacion = "Mes no válido"; } console.log(`El mes ${mes} corresponde a: ${estacion}`);

📝 Resumen de la Sesión

  • Los operadores de comparación (===, !==, >, <, etc.) nos permiten comparar valores.
  • Las estructuras if, else if y else nos permiten controlar el flujo del programa.
  • El operador ternario (condición ? valorSiVerdadero : valorSiFalso) es una forma concisa de escribir condiciones simples.
  • Es importante manejar todos los casos posibles en nuestras condiciones.
  • Las condiciones se evalúan en orden, por lo que debemos ser cuidadosos con el orden de nuestras condiciones.
